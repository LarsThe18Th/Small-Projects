#### FlipHz
------

Tool for switching MSX VDP to 50 or 60Hz

FlipHz.com => Flips MSX VDP from 50Hz to 60Hz, or from 60Hz to 50Hz  
50Hz.com => Switches MSX VDP to 50Hz  
60Hz.comm => Switches MSX VDP to 60Hz  
  
  

#### Notes
-----
Basic equivalent:  


50Hz:  

vdp(10)=vdp(10)or2

60Hz:  

vdp(10)=vdp(10)and253
