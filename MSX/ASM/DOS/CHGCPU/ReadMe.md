#### CHGCPU
-------

This tool changes the CPU mode of a MSX Turbo-r

usage: CHGCPU 0 / CHCCPU 1 /CHGCPU 2

0 = Z80 (ROM) Mode
1 = R800 (ROM) Mode
2 = R800 (DRAM) Mode
